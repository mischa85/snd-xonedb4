# snd-xonedb4

This is a driver for the Allen & Heath Xone:DB4 mixer.

Linux:

- [x] PCM out 8 channels
- [x] PCM in 8 channels
- [ ] MIDI out
- [x] MIDI in

Mac:

- [x] PCM out 8 channels
- [x] PCM in 8 channels
- [ ] MIDI out
- [ ] MIDI in
